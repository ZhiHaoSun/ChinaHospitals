产品需求文档 (PRD): 跨境医疗旅游 AI 智能规划系统 (MedTour AI)
版本: v1.0
日期: 2026年6月29日
目标用户: 拟来华进行医疗消费（如近视矫正、牙科种植、高端体检、医美等）的海外高净值及中产人群（主要包括新加坡、美国、马来西亚等国家）
1. 项目背景与痛点分析
中国医疗技术近年发展迅速，得益于庞大的临床病例数及耗材集中采购（VBP）政策，在眼科、齿科、高端体检以及部分前沿外科手术上展现出了极高的“价格套利”空间（性价比高、公立国际部效率极高）。
然而，海外患者（如新加坡、美国、马来西亚籍）来华就医面临以下严重痛点：
身份与挂号限制： 无法使用海外护照直接在微信小程序挂号，公立三甲医院国际部/特需门诊预约流程复杂。
信息不对称： 难以精准评估“医疗费用+跨境差旅（机票、酒店、签证）”的整体预算。
行程脱节： 医疗项目（如近视手术的术前检查、手术、术后第一天复查）与旅游行程难以完美缝合，导致时间浪费。
2. 产品愿景与定位
基于 Google Cloud Platform (GCP) 打造一套基于 Multi-Agent (多智能体) 架构的跨境医疗旅游智能规划系统。该系统通过多个具备专业垂直能力的 Agent 协同，为海外患者提供一站式、定制化的“医疗评估 + 差旅规划 + 精准预算评估 + 智能时间表生成”解决方案。
3. 系统架构与 Multi-Agent 设计 (基于 GCP)

4. 核心功能需求 (Functional Requirements)
4.1 用户输入与需求结构化
系统需支持多语言输入（英文、中文、马来文等）。
用户需勾选/输入：国籍、拟进行项目（如：全飞秒近视手术）、是否有内地身份证、期望出发时间、预算等级（经济/高端）。
4.2 智能医疗匹配与就医周期锁定
功能描述： Medical Agent 自动匹配目标城市的知名医院（例如眼科匹配广州中山眼科或上海五官科医院国际部）。
输出硬性指标： 必须明确锁定该项目的硬性时间窗（如：全飞秒需要至少保留3-4天，第1天检查、第2天手术、第3天复查）。
4.3 智能差旅与本地交通无缝缝合
功能描述： Travel Agent 根据医疗锁定的日期，反向推荐最佳往返航班，并利用 Google Places API 推荐距离医院步行 15 分钟以内、支持外宾入住（涉外资质）的酒店。
4.4 动态全额费用评估报告 (Financial Dashboard)
功能描述： Finance Agent 生成对比账单，包含：
中国医疗总预估费（含特需门诊挂号、术前检查、手术、基础药费）。
差旅总开销（机票 + 酒店 + 本地餐饮交通预估）。
海外本土对比价（对应新加坡/美国/马来西亚当地做同等手术的基准价）。
最终净节省金额量化。
4.5 智能可视化时间表生成 (Timeline Generator)
功能描述： 系统需输出一条以“小时”为单位的甘特图/时间轴线。
示例：Day 1 09:00 抵达浦东机场 -> Day 1 14:00 医院国际部术前检查 -> Day 2 09:00 手术 -> Day 3 09:00 术后第一天复查 -> Day 3 14:00 城市轻度外滩游。
5. 非功能需求 (Non-Functional Requirements)
数据隐私与安全 (Data Privacy)： 由于涉及海外用户的医疗意向和护照国籍信息，系统架构必须严格遵守 GDPR 及相关国家的数据隐私法律。在 GCP 上需启用 Cloud Key Management Service (KMS) 和 VPC Service Controls 进行数据隔离。
高并发与低延迟： 多智能体协同串行容易导致前端等待时间过长。Orchestrator Agent 必须支持部分任务的并行化（如医疗计费与差旅计费同步进行），整体生成报告时间控制在 15 秒以内。
接口健壮性： 中国本地医疗机构数据的变更（如挂号政策调整）需通过 RAG 实时更新，避免 Agent 基于老旧数据进行误导性规划。
6. 关键用户旅程示例 (User Journey Example)
用户： Sun, 新加坡籍，拟前往上海做“全飞秒 (SMILE Pro) 激光手术”，预算级别：高端特需。
MedTour AI 生成交付物：
医疗建议： 推荐上海市五官科医院国际部，预估医疗费 22,000 RMB (约 4,070 SGD)。提醒：由于是新加坡籍，可免签入境 30 天（依据最新免签政策）。
差旅方案： 推荐新加坡航空往返浦东机场（650 SGD）及医院周边涉外五星级酒店3晚（共 450 SGD）。
ROI 计算器： 新加坡本地全飞秒均价约 6,500 SGD。中国总开销（医疗+差旅）= 5,170 SGD。最终净省：1,330 SGD，且额外享受了一趟上海度假。
详细行程表： 输出包含航班落地接驳、医院特需通道面诊、手术及术后回新复查建议的完整 Timeline。
